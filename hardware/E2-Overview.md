# TFU-E2: Hardware Overview  
*TFU Engineering Division – E-Series Platform Hardware Specifications*

---

## Platform Description
The TFU-E2 platform is TFU’s compact, high-CRI, high-output utility light system based on the S2-class host.

All E2 variants follow the same design philosophy:
- High-CRI Nichia emitters  
- Field-serviceable S2 architecture  
- Tail-clicky simplicity  
- Rugged, consistent, testable  
- Beam patterns chosen for **task performance**, not lumen marketing

This page documents all current E2 variants.

# TFU-E2 (Standard Model)
*Baseline E2 platform — clean, bright, balanced.*

### **Hardware Profile**
- **Host:** Convoy S2+  
- **Emitters:**  
  - Nichia **219BT 4500K**  
  - Nichia **219F 5000K**  
- **Optic:** Carclo **10511** TIR  
- **Driver:** 5A Buck  
- **Battery:** 18650 (30Q / M35A)  
- **Clip:** Low-profile TFU clip  
- **Mode Group:** 1–10–35–100  

### **Beam Character**
- Medium flood with defined hotspot  
- Neutral, natural CRI  
- 219BT = warm-neutral + smooth  
- 219F = crisp + clear

---

# TFU-E2 Urban
*Indoor/urban optimized warm beam with maximal CRI and comfort.*

### **Hardware Profile**
- **Host:** S2+ profile  
- **Emitter:** Nichia **519A 3500K R9080**  
- **Optic:** Convoy **15° bead TIR**  
- **Driver:** 5A Buck
- **Clip:** Low profile black steel clip
- **Battery:** 30Q  
- **Mode Group:** 1–10–35–100  

### **Beam Character**
- Warm, zero-artifact flood  
- Perfect for walls, hallways, garages  
- Gentle edges with a defined working center  

---

# TFU-E2 Classic
*Triple-emitter, high-output “modern classic” flooder with soft edges and elite CRI.*

### **Hardware Profile**
- **Host:** Convoy S2  
- **Emitters:** **Triple Nichia 519A 4500K** on 20mm MCPCB  
- **Optic:** Carclo **10508** TIR (wide flood, soft center)  
- **Driver:** **5A Buck**  
- **Battery:** Molicel M35A / Samsung 30Q  
- **Clip:** Low-profile TFU clip  

### **Beam Character**
- Beautiful, even floody wall of light  
- Zero harsh hotspot, extremely comfortable indoors  
- High CRI with perfect color accuracy  
- Huge output for the size  
- “Soft flood incandescent-but-better” vibe

### **Role**
- Indoor utility  
- Photography, inspection  
- Close and mid-range task light  
- Anyone who wants a wide, uniform beam

---

# TFU-E2 Classic+
*The Classic on performance steroids.*

### **Hardware Profile**
- **Host:** S2  
- **Emitters:** **Triple Nichia 519A 4500K**  
- **Optic:** Carclo **10508** TIR  
- **Driver:** **8A Buck**  
- **Thermal Stack:** Upgraded paste + improved assembly torque + optional spring bypass  
- **Battery:** 30Q or P28A  
- **Clip:** Low-profile TFU clip  

### **Beam Character**
- Same perfect Classic wall-of-light beam  
- More horsepower  
- Better sustained brightness  
- Faster heat rise, but smoother regulation  

### **Role**
- Premium field/general-purpose  
- High-output indoor/outdoor  
- “TFU Classic, but unmistakably stronger”
