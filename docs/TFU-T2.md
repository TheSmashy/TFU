## Overview

The **TFU-T2** is a compact, rugged 14500/AA task light built for real-world utility.  
It excels as a lightweight field tool, comms-bag utility light, or general-purpose EDC.

_Primary mission:_ Deliver clean, regulated output with high color fidelity in a small, serviceable host.

## At a Glance

- **Emitter:** Nichia 519A 
- **CCT:** 5000K, High-CRI
- **Driver:** TFU 14500/AA 12-group linear driver
- **Battery Support:**  
  - **14500 Li-ion** (full performance)  
  - **AA NiMH** (safe, reliable fallback)  
  - **AA alkaline** (emergency only; low output + leak risk)
- **Default Mode Group:** **Group 8 ‚Äî 1% / 10% / 50%**  
  Tuned for use with **Vapcell F12** to keep thermals under control.
- **Optional High-Output Mode Group:** **Group 5 ‚Äî 1% / 20% / 100%**  
  Designed for **Vapcell H10**, with strict time limits (see below).

## Thermal & Output Warnings

The T2 is a compact light ‚Äî physics applies.

### Using Group 8 (F12 recommended)
1% / 10% / 50%  
- Excellent for field use  
- Stable thermals  
- No strict time limits

### Using Group 5 (H10 only)
1% / 20% / 100%  
- **100% is extremely high output for a T2-class host**  
- Expect rapid heat rise  
- Output will drop quickly as the driver protects the light

**TFU recommendation:**  
- Treat **100%** as a *burst mode*  
- **Maximum recommended ON time: ~45 seconds**

If it feels too hot to hold, stop.
