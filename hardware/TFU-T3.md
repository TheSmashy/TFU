## Overview

The **TFU-T3** is a compact tactical light built on the **Convoy S6 platform**, configured specifically for **rapid positive identification (PID)** and immediate maximum output.

This build pairs the **Luminus SFT40 6500K emitter** with a **potted 8A regulated buck driver**, producing a tight, high-intensity beam from the S6 reflector while maintaining efficient regulation.

The configuration is intentionally simple and purpose-built:

- **Forward clicky switch**
- **100% always first**
- **No mode memory**
- **Minimal mode set**

The light is designed so that the user can **immediately access maximum brightness for identification or threat illumination**, while still providing lower levels for extended use.

Thermal behavior is typical of a compact high-output thrower: **100% is a burst mode**, while **20% serves as the sustainable working level**.

## Driver Configuration

**Mode Group:** 5  
**Mode Order:** `100% → 20% → 1%`  
**Memory:** Off

This configuration ensures:

- **Immediate 100% output on activation**
- Reliable behavior under stress
- Minimal UI complexity

Half-press cycling allows rapid access to lower modes when needed.

---

## Beam Profile

The **SFT40 emitter** combined with the **S6 reflector geometry** produces a beam optimized for reach and identification:

- **Tight hotspot**
- Controlled spill
- Strong center intensity for mid-distance PID

The result is a **compact light capable of serious throw from a pocketable platform**.

---

## Thermal Behavior

At **8A drive current**, the SFT40 produces substantial output for a light of this size.

Operational guidance:

| Mode | Intended Use |
|---|---|
| **100%** | Immediate PID / tactical burst |
| **20%** | Sustained working output |
| **1%** | Administrative / low light tasks |

Extended operation on **100% will generate significant heat**.  
For most practical use cases, **20% provides a better balance of output and thermals**.
